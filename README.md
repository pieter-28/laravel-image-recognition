## Clone Project
```
git clone https://github.com/pieter-28/laravel-image-recognition.git
```

## Setup Project
```
composer install
```
## setup npm
```
npm install && npm run dev
```

## setup .env
### setup key
```
php artisan key:generate
```

## migrate
```
php artisan migrate:fresh
````

## Run serve
```
php artisan serve
```
